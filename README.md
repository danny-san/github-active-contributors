# Пайплайн GitHub Active Contributors
Этот пайплайн собирает статистику по GitHub-репозиториям для дашборда.\
Данные о репозиториях забираются через GitHub API, сохраняются в PostgreSQL, после чего агрегируются и загружаются в ClickHouse.\
После запуска пайплайна его единственный DAG с описанными выше действиями автоматически запускается каждый день в 0:00.

## Запуск пайплайна на Windows 10/11
1. Скачайте архив репозитория и распакуйте в любом удобном месте на вашем ПК. Либо скопируйте HTTPS-ссылку на странице репо, откройте терминал Git Bash в той директории, где хотите разместить приложение на ПК, и запустите команду:
   ```
   git clone <https-ссылка>
   ```
2. Убедитесь, что у Вас установлен WSL и Docker Desktop. Откройте терминал Git Bash в папке со скачанным репо. Запустите docker-контейнеры для приложения командой:
   ```
   docker compose up -d
   ```
   Дождитесь, пока закончится скачивание и запуск контейнеров*.

  *В Docker Desktop убедитесь, что в логах контейнера airflow_scheduler-1 появилась ASCII-надпись Airflow, а в контейнере airflow_webserver-1 начали появляться такие строки:
   ```
   127.0.0.1 - - [01/Jun/2026:11:58:43 +0000] "GET /health HTTP/1.1" 200 283 "-" "curl/7.88.1"
   ```
   Если этого не случилось, то, вероятно, контейнерам не удалось скачать необходимые библиотеки. Включите VPN и повторите пункт 2.

4. Откройте бразуер, перейдите в интерфейс Airflow по адресу: http://localhost:8080/ — и введите username и password "admin". Убедитесь, что в колонке Runs у вас цифра 1 в зеленом круге, а в колонке Recent Tasks также в зеленом круге цифра 3.
5. В DBeaver (или аналоге) создайте подключение с БД PostgreSQL с такими данными:\
  • Host: localhost\
  • Port: 5434\
  • Database: stats_db\
  • Username: postgres\
  • Password: postgres 
6. В DBeaver (или аналоге) создайте подключение с БД ClickHouse с такими данными:\
  • Host: localhost\
  • Port: 8123\
  • Database:\
  • Username: default\
  • Password: default

Теперь вы можете просматривать собранные сырые данные в БД PostgreSQL и агрегированную статистику в БД ClickHouse.
